# 🍵 AI采茶动作捕捉系统

基于MediaPipe的实时采茶动作分析与评分系统。

## ✨ 功能特点

- 🎮 **体验模式**: 趣味评分、等级称号、截图分享
- 📊 **效率模式**: 采摘计数、速度统计、数据分析
- ✅ **质控模式**: 动作规范检测、实时提醒
- 📚 **教学模式**: 标准动作演示、对比纠正
- 🏆 **传承模式**: 动作录制存档

## 🚀 快速开始

### 1. 安装依赖

```bash
cd tea-picking-ai
pip install -r requirements.txt
```

### 2. 运行程序

```bash
streamlit run app.py
```

### 3. 使用说明

1. 点击"开始检测"按钮启动摄像头
2. 将手部置于摄像头前，模拟采茶动作
3. 系统会实时分析动作并给出评分
4. 查看右侧面板获取反馈建议

## 🎯 采茶动作评分标准

| 评分项 | 权重 | 说明 |
|--------|------|------|
| 捏取姿势 | 40% | 拇指食指捏取的标准程度 |
| 手指姿态 | 30% | 其他手指的自然弯曲度 |
| 动作稳定 | 30% | 手部动作的稳定性 |

## 🛠️ 技术栈

- **AI框架**: MediaPipe (Pose + Hands)
- **界面**: Streamlit
- **图像处理**: OpenCV
- **数据处理**: NumPy, Pandas
